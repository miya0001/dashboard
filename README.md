# Dashboard for GitHub organization

[![Build Status](https://travis-ci.org/miya0001/dashboard.svg?branch=master)](https://travis-ci.org/miya0001/dashboard)


## Getting started

```
$ git clone git@github.com:miya0001/dashboard.git
$ npm install
```

Start a http-server:

```
$ npm start
```

Then build:

```
$ npm run build
```

## Automated testing

```
$ npm test
```
